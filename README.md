## Overview

This is the official website for **Patil Architectural and Interiors**, a full-service architecture and interior design firm based in Bagalkot, Karnataka. The site is a single-file, self-contained HTML webpage designed for fast loading, minimal dependencies, and seamless deployment on GitHub Pages.

Built and maintained by **[Kiran Kumar K](https://github.com/kirankumark-sec)**.

## Features

- **Hero Slideshow** — Full-screen image carousel showcasing architectural projects
- **Services Section** — Architectural Design, Interior Design, Construction/Turnkey, 3D Visualization
- **Portfolio Gallery** — Lightbox-enabled project gallery with captions
- **Architecture Showcase** — Annotated design philosophy section
- **Interior Showcase** — Curated interior design highlights
- **Testimonials Slider** — Auto-rotating client testimonials
- **Contact Section** — Integrated Google Form with direct client contact info
- **Custom Cursor** — Branded copper-tone cursor for desktop
- **Page Loader** — Animated intro loader on first visit
- **Scroll Reveal Animations** — Smooth entrance animations throughout
- **Fully Responsive** — Optimized for mobile, tablet, and desktop
- **Zero External Dependencies** — No frameworks, no CDN scripts, no backend required

## Deployment

This site is deployed automatically via **GitHub Pages** using the `main` branch.

Any push to `main` triggers a redeploy — the live site updates within 1–2 minutes.

To deploy your own copy:

1. Fork this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://<your-username>.github.io/<repo-name>/`
